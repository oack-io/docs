Alert channels define where notifications are sent when a monitor changes state.

## Supported channels

| Channel | Free | Pro | Business |
|---------|------|-----|----------|
| Email | Yes | Yes | Yes |
| Slack | - | Yes | Yes |
| Discord | - | Yes | Yes |
| Telegram | - | Yes | Yes |
| Webhooks | - | Yes | Yes |
| PagerDuty | - | Yes | Yes |
| SMS & Calls | - | - | Coming soon |
